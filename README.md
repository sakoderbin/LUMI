# LUMI — Live in Style

A fashion e‑commerce front-end for a fictional brand called LUMI. Built as a static site with HTML, CSS, and JavaScript — no frameworks, no build step.

## Pages

| Page | File | Description |
|------|------|-------------|
| **Home** | `index.html` | Hero editorial, product showcase, blog feed, customer reviews |
| **Shop** | `Go_to_shopping.html` | Catalog grid with 20 products (Women, Men, Bags, Shoes, Accessories, Perfume) + inline search |
| **Cart** | `cart.html` | Shopping cart with quantity controls and totals |
| **Checkout** | `checkout.html` | Checkout form stub |
| **Login** | `login.html` | Account sign‑in page |

## Features

- **Responsive** — desktop & mobile layouts (breakpoint at 860 px)
- **Hamburger menu** on mobile
- **Scrollable product reels** with arrow controls on the home page showcase
- **Inline search** that filters product names on the shop page
- **Wishlist bookmarks** on every product card
- **Utility bar** — free shipping, 30‑day returns, secure checkout
- **Custom typography** — Playfair Display (headlines), Cormorant Garamond (body), bold 12 px nav

## Run locally

Open `index.html` in any browser. All assets are local except Google Fonts and Unsplash hot‑linked images.

## Tech

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (DOM, inline search)
- [Unsplash](https://unsplash.com) for product and editorial photography
- [Google Fonts](https://fonts.google.com) — Playfair Display & Cormorant Garamond
